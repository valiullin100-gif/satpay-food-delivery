# Satpay Food Delivery 🍕

Сервис доставки еды в городе Сатпаев.

## 📋 Структура проекта

```
satpay-food-delivery/
├── frontend/          # React + TypeScript
├── backend/           # Node.js + Express + TypeScript
├── docs/              # Документация
└── docker-compose.yml # Docker конфигурация
```

## 🚀 Быстрый старт

### Требования
- Node.js 18+
- npm или yarn
- PostgreSQL 13+

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## 🏗️ Основные функции

- ✅ Регистрация и авторизация пользователей
- ✅ Просмотр меню ресторанов
- ✅ Добавление блюд в корзину
- ✅ Оформление заказа
- ✅ Отслеживание доставки
- ✅ История заказов
- ✅ Панель администратора для ресторанов
